Building a Magical 3D Button

Bet you can't click just once!

I had a neat realization recently: Buttons are the “killer feature” of the web. Every significant thing we do online, from ordering food to scheduling an appointment to playing a video, involves pressing a button. Buttons (and the forms they submit) make the web dynamic and interactive and powerful. But so many of those buttons are lackluster. They can trigger enormous changes in the real world, but they don't feel tangible at all. The feel like dull everyday pixels. In this tutorial, we'll build a whimsical 3D button:

Intended audience

This is an intermediate-level tutorial for front-end developers. It's focused on HTML/CSS, no JavaScript knowledge required. If you're relatively new to CSS transitions, I'd recommend reading “An Interactive Guide to CSS Transitions” first. Link to this heading

Our strategy

There's one main trick we'll use a couple times in this tutorial to create the illusion of a 3D button.

Paso a paso

Estructura del componente .btn => Este es el componente base que define y encapsula las variables y estilos generales como la estructura principal del botón. .btn_primary: Esta clase agrega estilos específicos para definir el tema principal del botón. Simplemente agregando otra clase, como .btn_secondary, puedes cambiar fácilmente el estilo del botón a un tema secundario. .btn_shadow: Representa la última capa del botón, la sombra. Utilizamos el pseudo-elemento :before o :after para lograr un posicionamiento que no afecte a los elementos hijos, permitiéndonos mover la sombra fuera de la caja principal sin afectar a los elementos front y back. .btn_back: Capa intermedia situada entre la sombra y la parte frontal del botón. Proporciona la estructura y los estilos de fondo del botón. .btn_front: La capa frontal del botón, que contiene el contenido visible del botón. Aquí es donde se define el estilo visual y el contenido del botón.

Utilizando una de las variables principales raise la qual maneja la elevacion en contexto 3D del botón, tecnicamente se le aplica a la base donde el alto (2d) total es la altura definida mas la elevacion 3d que queremos

Css

 
.btn{
  --height: 50px;
  height: calc(var(--height) + var(--raise)); 
  --raise: 100px;
}
​
.btn_shadow:before {
  height: var(--height); 
  margin-top: calc((var(--raise)/0.5)); 
}
​
.btn_back:before{
  height: var(--height); 
  margin-top: var(--raise);   
}

En términos sencillos, las tres capas del botón (shadow, back y front) son clave para lograr el efecto 3D. Al utilizar pseudo-elementos y técnicas de posicionamiento, podemos separar la sombra del botón principal sin afectar la disposición de los elementos visibles. Experimentar con estas clases te permitirá entender mejor cómo cada capa contribuye al diseño general del botón y cómo puedes personalizarlo según tus necesidades. ¡Diviértete explorando las posibilidades!
